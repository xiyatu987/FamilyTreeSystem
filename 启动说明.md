## 启动前准备

### 1. 安装PHP环境

**检测结果**：当前系统未安装PHP或PHP未添加到系统路径

**解决方法**：
1. 下载PHP 7.4：https://www.php.net/downloads.php
2. 安装并配置PHP环境
3. 添加PHP到系统环境变量

### 2. 安装Web服务器

推荐使用以下Web服务器之一：
- Apache
- NGINX
- IIS

### 3. 安装数据库

推荐使用MySQL数据库

## 快速启动指南

### 方法1：使用PHP内置服务器（推荐用于开发测试）

```bash
# 1. 安装PHP 7.4并配置环境变量
# 2. 进入webtrees目录
cd C:\Users\54143\Downloads\webtrees-2.2.4\webtrees

# 3. 启动PHP内置服务器
php -S localhost:8000

# 4. 在浏览器中访问
http://localhost:8000
```

### 方法2：使用本地Web服务器

1. 将webtrees目录部署到Web服务器的文档根目录
2. 配置虚拟主机指向webtrees目录
3. 启动Web服务器和数据库
4. 在浏览器中访问配置的URL

## 首次启动步骤

1. 打开浏览器访问系统URL
2. 跟随webtrees安装向导：
   - 选择语言（支持中文）
   - 配置数据库连接
   - 创建管理员账户
   - 创建家族树

3. 系统配置完成后，在控制面板中启用中国族谱管理模块

## 系统功能

安装完成后，您可以使用以下中国特色功能：

1. **字辈管理**：记录和管理家族字辈
2. **中式亲属关系**：使用中式亲属称谓
3. **宗族组织结构**：管理宗族分支和成员
4. **祠堂信息管理**：记录祠堂基本信息和活动
5. **迁徙与祖籍记录**：记录家族迁徙历史
6. **墓地信息管理**：管理祖先墓地信息
7. **家规家训**：记录和传承家族规则
8. **传统中式风格**：享受中式设计界面
