# 族谱管理系统操作手册

## 1. 系统概述

### 1.1 系统简介
族谱管理系统是一个专门为中国家族文化设计的在线管理系统，支持家族成员信息管理、中式亲属关系计算、字辈管理、祠堂管理、迁徙记录等特色功能。系统采用现代化的Web技术架构，提供直观易用的操作界面。

### 1.2 核心功能模块
- **家族成员管理**：完整的人员信息档案管理
- **字辈管理**：字辈库创建与自动分配
- **亲属关系查询**：中式亲属称谓计算
- **家族树可视化**：直观的家族谱系图展示
- **宗族管理**：组织结构与成员管理
- **宗族活动**：活动记录与照片管理
- **家规家训**：家族规则文档管理
- **祠堂管理**：祠堂信息与照片记录
- **迁徙记录**：家族迁徙路线与历史
- **墓地信息**：墓地位置与纪念管理

### 1.3 技术特色
- 响应式设计，支持桌面和移动设备访问
- 中式亲属关系算法，符合中国家族文化
- 可视化家族树，支持多代关系展示
- 照片和文档管理功能
- 数据导入导出支持

## 2. 系统登录与导航

### 2.1 用户登录
1. 打开系统主页 `http://localhost:8080`
2. 点击"登录"按钮进入登录页面
3. 输入用户名和密码
4. 点击"登录"按钮进入系统

### 2.2 主界面导航
登录后，系统主界面包含以下主要导航菜单：
- **首页**：家族树和最新动态
- **家族成员**：人员信息管理
- **字辈管理**：字辈库管理
- **家族树**：谱系图可视化
- **宗族管理**：宗族组织结构
- **宗族活动**：活动记录管理
- **家规家训**：家族规则文档
- **祠堂管理**：祠堂信息记录
- **迁徙记录**：家族迁徙历史
- **墓地信息**：墓地位置管理

### 2.3 移动端访问
系统采用响应式设计，在移动设备上访问时：
- 导航菜单会折叠为汉堡菜单按钮
- 表格和列表会自动调整为移动端友好的布局
- 触摸操作优化，支持手势操作

## 3. 家族成员管理

### 3.1 查看家族成员列表
1. 点击导航菜单中的"家族成员"
2. 系统显示家族成员列表
3. 默认显示族内人员（不含配偶）
4. 可勾选"显示全部人员（含配偶）"查看所有人员

### 3.2 添加新家族成员
1. 点击"添加新成员"按钮
2. 填写成员基本信息：
   - **姓名**：必填项
   - **性别**：男/女/其他
   - **出生日期**：格式：YYYY-MM-DD
   - **出生地**：可选填写
   - **宗族**：默认选中第一个宗族
   - **字辈**：选择所属字辈
   - **世代**：自动根据字辈设置
3. 选择家庭关系：
   - **父亲**：从下拉列表选择
   - **母亲**：从下拉列表选择
   - **配偶**：可选择现有成员或添加新配偶
4. 点击"保存"按钮完成添加

### 3.3 编辑成员信息
1. 在成员列表中点击"编辑"按钮
2. 修改需要更新的信息
3. **取消选择现有成员**：如果需要更换配偶，点击"取消选择"按钮清空当前选择
4. 点击"保存"按钮完成修改

### 3.4 删除成员
1. 在成员列表中点击"删除"按钮
2. 确认删除操作
3. 系统跳转到删除确认页面
4. 点击"确认删除"完成删除操作

### 3.5 成员信息筛选
- **默认筛选**：仅显示有宗族关系的族内人员
- **显示全部**：勾选"显示全部人员（含配偶）"查看所有人员（包括配偶）
- **搜索功能**：使用姓名、出生地等关键词搜索

## 4. 字辈管理

### 4.1 查看字辈列表
1. 点击导航菜单中的"字辈管理"
2. 系统显示所有字辈列表
3. 显示字辈名称、所属世代、排列顺序等信息

### 4.2 创建新字辈
1. 点击"添加字辈"按钮
2. 填写字辈信息：
   - **字辈名称**：必填项
   - **所属世代**：数字表示
   - **排列顺序**：系统自动设置为与所属世代相同
3. 点击"保存"按钮完成创建

### 4.3 编辑字辈
1. 在字辈列表中点击"编辑"按钮
2. 修改字辈信息
3. **注意**：排列排序字段已隐藏，系统自动设置为等于所属世代
4. 点击"保存"按钮完成修改

### 4.4 字辈分配
1. 在家族成员管理中创建或编辑成员时
2. 选择字辈后，世代信息会自动同步
3. 系统确保字辈与世代的一致性

## 5. 家族树可视化

### 5.1 查看家族树
1. 点击导航菜单中的"家族树"
2. 系统以树状图形式展示家族关系
3. 支持缩放和平移操作
4. 移动端自动调整为垂直布局

### 5.2 家族树交互
- **缩放**：鼠标滚轮或触摸手势
- **平移**：拖拽移动视图
- **详细信息**：点击节点查看成员详情
- **关系线**：显示父子、配偶关系连接

## 6. 宗族管理

### 6.1 创建宗族
1. 进入宗族管理页面
2. 点击"创建宗族"按钮
3. 填写宗族信息：
   - **宗族名称**：必填项
   - **宗族描述**：可选填写
   - **成立时间**：日期选择
4. 点击"保存"完成创建

### 6.2 管理宗族成员
1. 选择要管理的宗族
2. 查看成员列表
3. 添加或移除成员
4. 设置成员在宗族中的角色

## 14. 常见问题解答

### 14.1 登录问题
**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码

**Q: 无法访问系统？**
A: 检查网络连接，确认服务器正常运行

### 14.2 数据操作问题
**Q: 删除成员后可以恢复吗？**
A: 删除操作不可逆，建议删除前先备份数据

**Q: 如何修改错误的成员关系？**
A: 在编辑成员页面重新设置家庭关系

**Q: 字辈信息不匹配怎么办？**
A: 检查字辈设置，确保世代与字辈一致

### 14.3 显示问题
**Q: 家族树在手机上显示不全？**
A: 系统已优化移动端显示，可使用缩放功能

**Q: 如何筛选查看特定人员？**
A: 使用列表页面的搜索和筛选功能

### 14.4 功能使用问题
**Q: 配偶信息没有更新？**
A: 确保在添加/编辑时正确选择配偶信息

**Q: 如何取消已选择的现有成员？**
A: 在编辑页面点击"取消选择"按钮

**Q: 如何查看所有人员包括配偶？**
A: 勾选"显示全部人员（含配偶）"复选框
